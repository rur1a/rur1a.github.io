It is my first big project after I decided to be a Game Programmer and had internship developing Mobile Games. In this game I was responsible of full code: architecture, mechanics, different game logic and gameplay features. Especially I paid attention to build responsive, easy to understand and expandable architecture. For that I used simple Service Locator as DI Container, State Machine for manipulating and controlling all Game States and connect this in Game Entry Point. Also tried to manipulate assets using Resources folder and had plans to replace this with Addressable, but I didn’t have enough time for that.
